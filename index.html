SPECTRA Commands

Slash commands that run communities

Every command is role-gated and returns a rich Discord embed.

/ssu
Start a 5-minute voting session for Serious, Semi-Serious, or Casual roleplay mode.
/game
Announce a live game session with host info, mode, player count, and an Ask to Join button.
/deploy
Post a deployment announcement with host, co-hosts, mode selection, and optional banner image.
/announce
Send custom announcements to a channel with optional role mentions.
/delete_announce
Clean up announcements by message ID or the latest bot message in a channel.
/shutdown
Owner-only command to safely shutdown the bot instance.
